When Using the Tool

  • To delete very old tweets, please use the "Archive Version" at the bottom of the page
  • The developer does not collect any data whatsoever
  • When trying to delete large amounts, you may hit API limits. In such cases, the tool will stop, but you can try again after 15 minutes when the limit resets
  • "Replies" do not include self-replies
  • Date selection "from/to" includes the selected dates
  • Selecting "Create private list" will temporarily create a list visible only to you, which will be deleted upon completion. This increases the maximum deletion limit from 3000 to 12000
